Hilfe

Diese Seite zeigt den aktuellen Spielstand der Bulibot-Liga. Die Tabelle listet alle Spieler mit ihren Bulibots und den zugehörigen Details pro Spieltag. Unter der Tabelle können die zurückliegenden Spieltage durchblättert werden. Daten aus vorherigen Saisons sind über den Header wählbar. Während Bundesliga-Spiele ausgeragen werden, werden die Daten regelmäßig aktualisiert und sind somit nahezu live verfügbar. Die Verfügbarkeit hängt hier jedoch stark von der Aktualität der Daten in openligadb ab.

Punktevergabe

Vergleichbar mit bekannten Tippspielen wird jedes errechnete Bulibot Ergebnis mit dem tatsächlichen ergebnis der Bundesliga Partie verglichen. Dabei werden die folgenden möglichen Situationen wie aufgeführt mit Punkten bewertet:

Situation Punkte
Exaktes Ergebnis 4
Relatives Ergebnis 3
Korrekte Tendenz 2
Falsche Tendenz 0
Fehlerhafte Berechnung 0

Platzierung

Die Platzierung eines Bulibots wird primär aber nicht aussschließlich auf Grundlage der erreichten Punkte berechnet. Die Kriterien werden in der folgenden Reihenfolge ausgewertet, sind für zwei Bulibots die Kriterien identisch, so wird das nächste aus der Liste verwendet:

  1. Punkte
  2. Exakte Ergebnisse
  3. Relative Ergebnisse
  4. Bulibot Versionen*
  5. Fehlerquote*

*Bei diesen Kriterien werden niedrigere Werte als besser betrachtet.
